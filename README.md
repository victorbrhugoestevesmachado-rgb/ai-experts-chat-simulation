## 🧠 Sobre o Projeto
Este projeto utiliza Inteligência Artificial para simular debates entre especialistas renomados da área, com base em conteúdos reais.

O conteúdo foi desenvolvido com apoio do NotebookLM, utilizando vídeos e artigos como fonte.

## 🎯 Objetivo
Explorar diferentes perspectivas sobre Inteligência Artificial:
- Presente e futuro
- Impacto nos empregos
- Uso no dia a dia

## 💬 Diferencial
Simulação de debates entre especialistas, promovendo aprendizado ativo e pensamento crítico.
